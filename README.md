# 🧠 Pokémon Data Analysis & Dashboard

## 📌 Overview

This project analyzes Pokémon data to uncover patterns in stats, types, and legendary classification using Python and builds an interactive dashboard using Streamlit.

## 🚀 Features

* Advanced Exploratory Data Analysis (EDA)
* Feature Engineering (Total Stats, Role Classification)
* Hypothesis Testing
* K-Means Clustering (Segmentation)
* Interactive Dashboard with Filters

## 🛠️ Tech Stack

* Python (Pandas, NumPy)
* Visualization (Matplotlib, Seaborn, Plotly)
* Streamlit (Dashboard)
* Scikit-learn
